# eCommerce

### Project Setup
```
1. git clone https://github.com/Prince3855/eCommerce.git
2. cd eCommerce
3. npm install
```

### Run Project
```
npm start
```

### APIs Details

#### Admin(Seller) Dashboard APIs
1. login
2. register
3. create product
4. delete product
5. update product
6. get all products of seller

#### Customer or Main Applications APIs
1. login
2. register
3. view all products
4. place order
5. get all orders

#### superAdmin
1. create, remove category
